Tool Choices:

There are many ways to create a block puzzle like the one shown on the home age, and some ways are a lot easiier than others.

You could use all hand tools: a hand saw (such as a coping saw), chisels, and a hand plane. However, there are many cuts that have to be made on this project, so if you decide to used hand tools, plan on this project to take a lot of time.

A more time efficient way of creating this project is to use power tool, either a router or a table saw. Of These two choices, I would recommend a table saw. Since the individual pieces that you will be making are small, if you decide to use a router, you should also require a router table. It could become dangerous making these router cuts without using a a router table with your router.

There is another reason for using a table saw over a router. The cuts that you will be making need to be slightly larger that what is indicated in the plans (around 1/32nd of an inch wider and deeper) for other pieces to solve the puzzle. Other wise the pieces will be too tight and you may not be able to fit al the pieces together.

If you have not guesses already, my preference is to use a table saw. However, this recommendation has one caveat. You need to use a dado blade with your table saw and dado blades can be expensive. Using a dado set of blades with your table saw will save a lot of time getting all the pieces to fit together well.


Materials:

The thickness of the wood provided in this plan calls for the wood to be 3/4 of an inch in thickness. You should be able to find this thickness in all home improvement centers or lumber yards. You may not have a very large selection of types of wood at many of these places. For example: Home Depot and Lowes used to sell a decent selection of walnut and maple along with oak and popular. Now, they only sell oak and popular. It is my opinion that the finished product looks better if you can find different types of wood for the pieces that you will make.
